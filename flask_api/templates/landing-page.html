About

Ebisu Flashcards is a new flashcard application centerd on the new Ebisu algorithm, developer by fasiha. It features a much easier to use interface than Anki, and aims at providing the same amount of features, especially language learning-oriented ones.

It's fundamentally different from the classic SM2/Anki approach to spaced repetition, because it's completely based on probability theory and on the exponential decay of memory [instert author and paper here].
